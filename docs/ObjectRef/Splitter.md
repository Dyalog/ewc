# Splitter

The EWC implementation of Splitter has some degree of support for:

 |   Event  |   SplitObj1  |   Style    |         |
 |  *Posn   |   SplitObj2  |   Visible  |         |

* indicates that the property can change after it has been set.

Supported events:

 |  EndSplit  |            |            |           |
