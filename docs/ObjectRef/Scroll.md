# Scroll

The EWC implementation of Scroll has some degree of support for:

 |   Align   |   HScroll  |   Size     |   Visible |
 |   Attach  |   Posn     |  *Thumb    |           |
 |   Event   |   Range    |   VScroll  |           |

* indicates that the property can change after it has been set.

Supported events:

 |  Scroll  |          |          |         |
