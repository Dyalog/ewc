# Combo

The EWC implementation of Combo has some degree of support for:

 |   Attach  |   Items  |  *SelItems  |  *Text    |
 |   Event   |  *Posn   |  *Size      |   Visible |

* indicates that the property can change after it has been set.

Supported events:

 |  Select  |          |          |         |
