# SubForm

The EWC implementation of SubForm has some degree of support for:

 |   BCol     |   Flex     |  *Size    |   Visible |
 |   Border   |   Picture  |   Styles  |           |
 |   Caption  |  *Posn     |   TabObj  |           |

* indicates that the property can change after it has been set.

Supported events:

 |  Configure  |             |             |            |
