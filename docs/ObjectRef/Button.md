# Button

The EWC implementation of Button has some degree of support for:

 |   Align    |   Event    |  *Size   |   Styles  |
 |   Attach   |   Picture  |  *State  |   Visible |
 |   Caption  |  *Posn     |   Style  |           |

* indicates that the property can change after it has been set.

Supported events:

 |  KeyPress  |  Select  |            |           |
